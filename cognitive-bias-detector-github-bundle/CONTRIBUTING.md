
# Contributing

Thanks for your interest!

## How to run locally (Windows)

```cmd
py -m venv .venv
.venv\Scripts\activate
py -m pip install -r requirements.txt
py app.py
```

## Code style

- Python 3.10+
- Keep functions small and documented.
- If you add dependencies, update `requirements.txt`.
